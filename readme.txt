== Description ==
EasyChic is a multipurpose WordPress theme perfect for blogs, portfolios, and business websites.
The responsive design ensures beautiful display across all devices.

Key Features:
* Easy customization with WordPress Customizer
* Full Block Editor Support
* Fast Loading
* Responsive Design

== Installation ==
1. Go to "Appearance" → "Themes" in the admin panel
2. Click "Add New"
3. Enter "Easychic" in the search form
4. Install
5. Click Activate
6. Go to the Easychic menu in your admin panel　

= Where can I customize the theme? =
Go to the Easychic menu in your admin panel and customize it.
You can customize various settings from Appearance → Customize in the admin panel.
